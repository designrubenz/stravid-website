--- layout: english_post title: Achtung die Kurve with HTML5 Canvas ---

Finally I can announce that Achtung die Kurve is released!

Achtung die Kurve is a little project I did with my friend Mathias Paumgarten. We wanted to port the old C++ / Flash game to the browser using only JavaScript and the HTML5 Canvas element.

The game description and the "How to play" instructions can be found on the project page!

If you are interested in the source code the GitHub repository is the place to be, we appreciate any comments and suggestions.
